# GainYourMuscle — Frontend

React 19 + Capacitor 8 (Android + PWA)

## Scripts

```bash
npm start          # Desenvolvimento local
npm run build      # Build de produção
npm run cap:sync   # Build + sync com Android
npm run cap:open   # Abre o Android Studio
```

## Deploy

O deploy é automático via **Vercel** a cada push na branch `main`.
